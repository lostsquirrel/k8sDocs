入门

本节列举了几种不同的搭建和运行 k8s 的方式。 在安全 k8s 时，选择安装方案基于: 易于维护， 安全，控制，可用资源，集群运维和管理的经验要求等

你可以将 k8s 部署在本地主机，云主机，本地数据中心, 或选择一个由托管的 k8s 集群。 也可以基于云供应商或 bare metal来搭建自定义解决方案
